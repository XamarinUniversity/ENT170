Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin.Forms development environment and a simulator, emulator, or a device on which to run the exercises. Please see the setup page if you need help installing the Xamarin development environment.

Overview

In this class you will see how to store data locally in your mobile applications. You will encrypt the data before storing it and decrypt it after retrieval. All the key operations will be done in a cross-platform way so you can share the code across all the platforms you target.

Objectives

  1. Store secrets with Xamarin.Auth
  2. Introduce PCLCrypto
  3. Hash passwords with PCLCrypto
  4. Encrypt/decrypt data with PCLCrypto

Lecture

ENT170 - Securing Local Data.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Store user data with Xamarin.Auth
    Exercise
    In this exercise, we will securely store user-entered data.
  2. Hash and validate a password
    Exercise
    In this exercise, we will use hashing to increase security when we store and validate a user's password.
  3. Encrypt and decrypt data
    Exercise
    In this exercise, we will encrypt personal data being stored locally.
